# Arduino knihovny

Všechny knihovny jsou dostupné přes Arduino Library Manager (Sketch → Include Library → Manage Libraries).

| Knihovna | Poznámka |
|----------|----------|
| DallasTemperature | Čtení DS18B20 čidel přes OneWire |
| OneWire | Závislost DallasTemperature |
| Ds1302 | RTC hodiny reálného času |
| TX07K-TXC | Dekódování bezdrátových teplotních čidel TX07K-TXC |
| MQTTESP8266 | MQTT klient (MQTTClient) + ovladač ESP8266 (EspDrv) |
| LiquidCrystal_I2C | I2C LCD displej 16×2 |
